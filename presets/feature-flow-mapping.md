# Feature Flow Mapping Preset

Read the codebase and map the end-to-end flow for the requested feature.

Return:

1. Entry points.
2. Services involved.
3. External dependencies.
4. Events published or consumed.
5. Database writes.
6. Failure points.
7. Retry behavior.
8. Logs and observability gaps.
9. Security-sensitive data involved.

Include concrete file references and note areas where behavior is inferred rather than explicit.
