# Claude Allow Permissions

Common safe local development actions:

- Read, list, and search files.
- Inspect git status, diffs, branches, and logs.
- Run tests, lint, format, typecheck, build, and local dev servers.
- Use `npm`, `pnpm`, and `yarn` for project scripts.
- Use `gh` for non-destructive GitHub inspection.
- Use `docker` and `docker compose` for local development.

Ask before destructive operations, production access, secrets, auth changes, infrastructure changes, payment or billing behavior changes, database migrations, Docker prune/delete commands, pushing, merging, releasing, or deploying.
