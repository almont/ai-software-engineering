# Claude Allow Permissions

Common safe local development actions:

- Read, list, and search files.
- Inspect git status, diffs, branches, and logs.
- Run tests, lint, format, typecheck, build, and local dev servers.
- Use explicit `npm`, `pnpm`, and `yarn` scripts such as test, lint, typecheck, build, and dev.
- Use `gh` for non-destructive GitHub inspection.
- Use `docker` and `docker compose` for local inspection such as `ps` and `logs`.

Ask before broad package manager script prefixes, Docker lifecycle commands, destructive operations, production access, secrets, auth changes, infrastructure changes, high-risk business workflow changes, database migrations, Docker prune/delete commands, pushing, merging, releasing, or deploying.
